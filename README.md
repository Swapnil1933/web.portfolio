# Swapnil Nikhij — Web Portfolio

**🌐 Live Site: [https://swapniltech.me](https://swapniltech.me)**

Personal portfolio website for Swapnil Nikhij, Data Analyst & CS Engineer.

## About

Built as a single-page portfolio with smooth scroll-reveal animations, custom cursor, preloader, and a dark minimal design.

## Sections

- **Hero** — Introduction with parallax depth effect
- **About** — Background, detail list, and social links
- **Skills** — Tech stack (Python, SQL, Power BI, Pandas, NumPy, Scikit-learn, etc.)
- **Projects** — Jharkhand Village Directory Dashboard, Customer Satisfaction Analysis
- **Experience** — Internship & hackathon achievements
- **Certificates** — Oracle Cloud Infrastructure certifications
- **Education** — Academic history
- **Contact** — Contact form and direct links

## Tech Stack

- Vanilla HTML / CSS / JavaScript
- [GSAP](https://gsap.com/) + [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/) for scroll-linked animations
- Google Fonts (Syne, DM Mono)
- Hosted on **GitHub Pages** with custom domain `swapniltech.me`
